# Simple Scroll

Responsive horizontal scrolling.

###Usage:

- `bower install simple-scroll` or [download it](https://github.com/ryanwalters/simple-scroll/archive/0.1.2.zip)
- `$('.your-items').scroll()` on a list of items styled however you want
